# 🚗 Motor Insurance Policy Management System

## 📖 Project Overview

The **Motor Insurance Policy Management System** is a MySQL-based database project developed to manage motor insurance operations efficiently. It provides a centralized system for maintaining customer information, vehicle details, insurance quotes, policies, payments, claims, endorsements, and renewals.

This project demonstrates relational database design and advanced SQL concepts using real-world business scenarios.

---

## 🎯 Objectives

- Design a relational database for motor insurance management.
- Manage customer, broker, and vehicle information.
- Generate insurance quotes and policies.
- Track payments and insurance claims.
- Handle policy endorsements and renewals.
- Demonstrate advanced SQL concepts through business scenarios.
